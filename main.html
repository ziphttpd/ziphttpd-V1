はじめに

１．ZipHttpd はローカルで動かす WEB サーバです。

HTML や画像などのコンテンツを zip に圧縮したファイルを用います。
なお、自動で解凍されますので利用者は意識する必要はありません。

２．ZipHttpd はファイルを解凍するだけで使えます。

レジストリなどを使いませんので、インストールは不要です。

３．ZipHttpd では容易にドキュメントを配布可能です。

無料のリポジトリサービスに登録されているドキュメントは、クリックしてダウンロードできます。

４．ZipHttpd では大容量のデータを保存できます。

保存するために JavaScript から呼び出せる WebAPI を実装しています。
数十MBを超えるような大容量のデータもブラウザに依存せずに使用できます。

５．ZipHttpd は多数の OS で利用可能です。

ZipHttpd はGo 言語で記述しています。
Go言語は Windows/Linux/iOS などで実行ファイルを作成できます。
標準ライブラリのみで記述していますので、ビルドすればそのまま利用可能です。
将来的には iPhone/Android などのモバイル端末での実行も検討しています。
